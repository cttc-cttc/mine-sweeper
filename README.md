# 웹 지뢰찾기 게임

### 개발도구
- Visual Studio Code

### 기술스택
- Javascript, HTML, CSS

### 프로젝트 개요
- 개발기간 : 2022.07 ~ 2022.07 (1주)
- 개발인원 : 1명

### 주요 업데이트 내용
- 게임 내 이미지를 윈도우 이모티콘 사용 -> 이미지 사용으로 변경
- 게임 로직을 1D -> 2D 형식으로 변경
- 클릭한 곳 기준 주변 8방향 검사 로직을 메소드화
